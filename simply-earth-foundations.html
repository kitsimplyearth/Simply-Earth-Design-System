4pt Grid
▦
4pt Grid System
All spacing values are multiples of 4px — the base unit. Micro gaps (4px, 8px, 12px) use the smallest steps; layout-level spacing (32px+) uses larger multiples. This ensures consistent visual rhythm across all components.
