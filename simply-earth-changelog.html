v1.7.0
Apr 6, 2026
Hero 3 Real Media, Arc Curves & Component Detail Page
Updated the Hero 3 (Recipe Box) block with a live video and real product image, added arc curves matching the simplyearth.com design, renamed and redescribed the block, and launched a new Component Detail page (simply-earth-component.html) with preview iframe, export CTAs, overview, component metadata, and related components — plus a "View Details →" button in the preview modal.
Updated Block renamed from "Hero – Video 58/42 Split" to Hero 3 — description updated to match the actual block anatomy (split-layout, category label, benefit bullets, CTA + urgency sub-label, social proof)
Updated Hero 3 video placeholder replaced with real <video autoplay muted loop playsinline> — source: April2026Montagev111.mp4, poster fallback: recipe-box.webp (both from images.simplyearth.com CDN)
New Arc curves added to Hero 3: .rb-hero-curve-right (desktop — SVG bezier arc on the right edge of the video panel, cream-filled) and .rb-hero-curve-bottom (mobile — convex arc transitioning to cream content card), matching the .curve-bottom pattern on simplyearth.com
New simply-earth-component.html — Component Detail page. URL-param driven (?id=rb-hero). Includes: live block preview via srcdoc iframe, Desktop / Tablet / Mobile viewport switcher, Copy for Figma + Copy HTML export buttons, Overview description, Component Details table (category, collection, tags, responsive, release, author), and Related Components grid (4 same-page blocks)
New "View Details →" button added to the preview modal footer in index.html — links to simply-earth-component.html?id={blockId}, dynamically set when any block is opened
Updated settings.local.json — added allowed Bash permissions: npx puppeteer:*, node -e ":* (both quote styles), netlify --version
Updated Hero 3 — Recipe Box label icon replaced: 📦 emoji swapped for the real recipe-box.webp product image (48×48px, object-fit: contain, 6px padding, 1px cream border)
Updated Hero 3 card thumbnail updated to use real recipe-box.webp poster image on the 58% video side with a cream gradient fade, replacing the dark teal gradient placeholder
Updated Component Detail page layout — max-width widened to 1520px, iframe outer padding removed so the block preview renders edge-to-edge on desktop. Breakpoints corrected to system values: sidebar collapses at 1024px, single-column stacks at 768px
v1.6.0
Mar 25, 2026
Layout & Nav Color Updates
Container max-width expanded to 1325px for desktop, mobile padding tightened to 16px, and secondary teal color removed from nav backgrounds — navbars now use light neutral backgrounds with balanced dark text.
Updated Container max-width updated from 1248px to 1325px across all container types — .container, .rb-container, .sp-container, .cat-container — and all --container-max tokens
Updated Mobile container padding reduced from 20px to 16px (base/default — tablet and desktop padding unchanged at 40px and 60px)
Updated Sub-nav background changed from var(--color-secondary) (#375D5A) to #F8F8F8 with a #E8E8E4 border — link color updated from white to #434343, active state uses teal bottom border accent
Updated Color rule clarified: #375D5A is no longer permitted on nav backgrounds — restricted to secondary buttons and subtle section backgrounds only
Updated Design Foundations page updated — swatch description, usage rules chip, and CSS comment all reflect the new nav color restriction. Nav background rule added: use #FFFFFF or #F8F8F8 with #434343 text
